Course Description


Design, analysis, implementation and comparison of efficient data structures for common abstract data types.

  • Priority queues: heaps and mergeable heaps.
  • Dictionaries: balanced binary search trees, B-trees, hashing.
  • Amortization: data structures for managing dynamic tables and disjoint sets.

Data structures for representing graphs and graph searches.


Prerequisite:

CSCB36H or CSCB38H and [CGPA 2.75 or enrolment in a CSC subject POSt]



Exclusive:

CSC263H, CSC265H, CSCC78H
